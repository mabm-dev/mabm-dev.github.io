# Portfolio de Miguel Ángel Burgos

Portfolio web estático, responsive y preparado para publicarse gratuitamente.

## Archivos

- `index.html`: contenido principal.
- `styles.css`: diseño visual y adaptación móvil.
- `script.js`: menú, animaciones y navegación activa.
- `CV-MiguelAngelBurgos.pdf`: currículum descargable.
- `assets/favicon.svg`: icono del sitio.

## Abrir en tu ordenador

Haz doble clic en `index.html`. También puedes abrir la carpeta desde Visual Studio Code
y utilizar la extensión **Live Server**.

## Publicar con GitHub Pages

1. Crea un repositorio público, por ejemplo `portfolio`.
2. Sube todos los archivos de esta carpeta a la raíz del repositorio.
3. En GitHub, entra en **Settings → Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Selecciona la rama `main` y la carpeta `/root`.
6. Guarda los cambios. GitHub mostrará la dirección pública del portfolio.

También puedes nombrar el repositorio `mabm-dev.github.io`. En ese caso, la dirección será:

`https://mabm-dev.github.io`

## Personalización rápida

- Para cambiar textos, edita `index.html`.
- Para cambiar el diseño, edita las variables situadas al principio de `styles.css`.
- Para añadir una fotografía, guarda la imagen dentro de `assets` y añádela en la sección
  principal del HTML.
- Cuando tengas demos públicas, sustituye o añade enlaces junto a los repositorios.

## Recomendación

Antes de publicarlo, comprueba que los enlaces de GitHub, LinkedIn, correo y descarga del CV
funcionan correctamente.
